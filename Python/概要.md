## Python新手入门教程

```
Python是一门及其简单且广泛使用的脚本语言。Python的教程非常多，其中Python官方的文档就是一个非常好的教程。另外，也有很多其他教程，其实随便选一本看就可以入门。本篇文档的作用主要在于引导读者在完全不懂Python的情况下，如何快速的入门。当然在后面的高级特性上也会给出一些我自己的理解，以便自己记忆。
```

#### Python的开发环境
```
Python 是一种跨操作系统的语言，而且可以不做编译即可运行。因此无论是在Windows、Linux还是Mac都可以使用Python进行开发。另外Python的编辑器个人建议选择PyCharm，就像微软的VS一样，PyCharm不仅仅可以作为编辑器使用，而且还可以作为编译器等使用，非常方便。
附：PyCharm官方网址，https://www.jetbrains.com/pycharm
```
#### Python版本选择和库的安装
```
pip和pyvenv
Python目前有Python2和Python3，个人强烈建议用Python3，因为Python官方已经放弃维护Python2了。另外Python在开发之前，不可或缺的就是安装pip，pip可以进行依赖库的管理，另外pyvenv也非常重要，可以进行虚拟环境的隔离。
附：
	pip：https://pip.pypa.io/en/stable
	pyvenv：https://docs.python.org/3/library/venv.html
```
